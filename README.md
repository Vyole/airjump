# airjump
 Simple and small project about user and tickets management using MySQL trying to learn about connections pools.
 
 This program has two modules: tickets and users.
 
 User Module:
 - Create an user.
 - Delete an user.
 - Edit an user.
 
 Tickets Module:
 - Buy Ticket.
 - Remove Ticket.
 - Edit Ticket.
